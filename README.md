# Auth Extractors

أدوات استخراج التوكنز. مباشرة، بدون لف ودوران.

## discord-login.py
**الهدف:** إثبات أن نظام "التحقق" في ديسكورد مجرد واجهة.
**الحقيقة:** لا يوجد أمان حقيقي في عملية تسجيل الدخول؛ بمجرد إدخال بيانات صحيحة، يخرج التوكن (Token) مباشرة. السكربت يقطف التوكن من الاستجابة (Response) فور توفره.

## Bluesky_login.py
**الهدف:** استخراج Session Token الخاص بحسابات Bluesky. 
**الحقيقة:** فحص مباشر لكيفية تعامل الـ API مع الجلسات. لا تعقيدات، فقط استخراج التوكن.

---

## ملاحظات
* **التوكن = الحساب:** أي كلام غير هذا هو كذب. بمجرد امتلاك التوكن، الحساب تحت تصرفك.
* **الأداء:** السكربتات خفيفة، للأغراض التحليلية فقط.
* **المسؤولية:** أنت المسؤول عن استخدامك لهذه الأدوات.
