# Auth Extraction & API Analysis Tools
## Developed by: xxxxxthefox

أدوات لتحليل عمليات المصادقة (Authentication) في التطبيقات. الهدف هو تجاوز واجهات المستخدم الوهمية والوصول المباشر للبيانات التشغيلية عبر استجابات الخوادم.

---

## 1. discord-login.py
يستهدف `https://discord.com/api/v9/auth/login`.

* **نظام النتاج:** يرجع الخادم كود `200` مع ملف `JSON` يحتوي على بيانات الجلسة الكاملة.
* **ماذا يمكن استخراجه:**
    * **User ID:** المعرف الفريد للحساب.
    * **Access Keys:** مفاتيح الدخول التي تمنح صلاحية التحكم الكامل.
    * **User Settings:** تفضيلات المستخدم (اللغة، الثيم).
* **الحقيقة التقنية:** "التحقق" في ديسكورد كذبة تقنية. بمجرد إدخال بيانات صحيحة، يمنحك السيرفر صلاحية الوصول فوراً دون أي قيود.

## 2. Bluesky_login.py
يستهدف `https://bsky.social/xrpc/com.atproto.server.createSession`.

* **نظام النتاج:** يرجع الخادم كود `200` مع `JSON` يحتوي على كامل بيانات الهوية والجلسة.
* **ماذا يمكن استخراجه:**
    * **Session Keys:** مفاتيح الوصول وتجديد الجلسة.
    * **DID:** الهوية اللامركزية للمستخدم.
    * **Account Info:** البريد الإلكتروني، الاسم المستخدم (Handle)، وحالة الحساب.
* **الهدف:** استخراج صلاحيات الجلسة الكاملة لإجراء عمليات فحص برمجية.

---

## المبدأ التقني
* **القدرة التشغيلية:** الأدوات مصممة لربطها بقوائم (Lists) من الحسابات لإجراء عمليات فحص جماعي (Bulk Checking) أو تخمين بيانات الدخول (Brute Force).
* **النتيجة:** امتلاك مفاتيح الوصول يتيح أتمتة التعامل مع الحسابات وتجاوز أي حماية يزعمها التطبيق.
